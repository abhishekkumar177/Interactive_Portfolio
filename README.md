# üöÄ Abhishek Kumar's Interactive Portfolio

This repository contains the source code for an **interactive, modern, and dynamic personal portfolio** website built with a focus on cutting-edge design and user experience. It features a dark/light theme toggle, engaging scroll-based animations, and a responsive layout to showcase the profile owner's skills, projects, and professional experience.

-----

## ‚ú® Features

  * **Dark/Light Mode Toggle:** A fully functional theme switcher that applies a distinct color palette to the entire site.
  * **Interactive Liquid Background:** A mesmerizing canvas background animation using JavaScript to create moving blobs and mouse-trail particles (liquid ether effect).
  * **Dynamic UI Elements:**
      * **Electric Border Effect:** Animated conic gradient borders that appear on hover for the profile picture and project cards.
      * **Scroll Float Animation:** Subtle vertical parallax effect on project cards as the user scrolls.
      * **Scroll Reveal Animation:** Experience items fade in when they enter the viewport using the Intersection Observer API.
  * **Skills Tab Switcher:** An organized section allowing users to switch between skill categories (Frontend, Backend, ML, Cloud).
  * **Responsive Design:** Optimized layout for all screen sizes, from mobile devices to large desktop monitors.
